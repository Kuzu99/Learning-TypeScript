# Practice-TypeScript

## Getting Started

- clone

```shell
git clone git@github.com:Kuzu99/Practice-TypeScript.git
```

- コンテナの起動

```shell
docker-compose up -d
```

- コンテナに入る

```
docker exec -it practice-typescript /bin/sh
```

- コンパイル

```shell
$ tsc 1-HelloWorld/hello.ts
```

- 実行

```shell
$ node 1-HelloWorld/hello.js
Hello World
```
